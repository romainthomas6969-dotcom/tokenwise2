Definition

C est quoi un token ?

Un token est le plus petit morceau de texte qu une IA peut traiter. Ce n est ni un mot entier, ni une lettre. 1 token correspond environ a 3/4 d un mot en anglais, un peu moins en francais.

Comment une phrase est decoupee en tokens
"Bonjour, comment puis-je vous aider ?" = 9 tokens

Facturation

Input vs Output

Toutes les IA facturent separement les tokens que tu envoies (input) et les tokens qu elles te renvoient (output). L output coute 3 a 8x plus cher que l input.

Tokens INPUT

Ta question, le document, le contexte. L IA le lit. Moins couteux.

Tokens OUTPUT

La reponse generee. L IA la cree token par token. 3 a 8x plus cher.
